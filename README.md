# shop-microservices
developed code repo


This repository includes microservices for the shop platform, and the services are:
•	Frontend
•	Product-service
•	Order-service
Each service is containerised using Docker.


## Local ports
•	Frontend : 5173
•	Product-service : 3001
•	Order-service : 3002

## Run locally
# product-service
cd product-service
npm install
npm start

# order-service
cd order-service
npm install
npm start

# frontend
cd frontend
npm install
npm run dev
